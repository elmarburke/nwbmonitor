NWBMonitor
==========

Ein alternatives Twitter-Frontend zum [Abfahrtsmonitor der Nordwestbahn](http://www.nordwestbahn.de/de/verkehrsmeldungen/abfahrtsmonitor.html).

Ohne Datenbank, ohne Zwischenspeicher und dadurch Echtzeit.
